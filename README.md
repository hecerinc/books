[#](#) Book Database

This is an attempt to catalog all the electronic books (PDFs & ebooks) I have and make a small database.


## Run

```
$ npm run start
```


Fields:

- Name
- Author
- ISBN
- NameofFile.pdf
- Link to book
- File size
